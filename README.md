# QTCaller

### build

```bash
git clone https://github.com/lizi-garden/qt-caller.git
cd qt-caller
cmake -S . -B build
cmake --build build
./build/qtCaller
```

If you build by qtcreater, please copy the namelists.txt to your build directory.

You can run these commands whichever system you use in.
